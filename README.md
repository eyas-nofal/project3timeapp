# Flutter Date & Time App

A simple Flutter app that displays the **current date and time**, updating every second.  
The time is shown in **12-hour format** with AM/PM.

---

## Features

- Displays the **day of the week** (Monday, Tuesday, …)  
- Displays the **month name** (January, February, …)  
- Shows the **day of the month and year**  
- Displays the **current time** in **12-hour format** with **AM/PM**  
- Updates **every second** automatically
