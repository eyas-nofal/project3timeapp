Date & Time App

A simple Flutter app that displays the current day, date, and time, updating every second.
Uses the intl package for formatting dates and times in a clean and professional way.

Features

Shows current day of the week (e.g., Monday, Tuesday).

Shows current date in the format: Month day, year (e.g., December 23, 2025).

Shows current time in 12-hour format with AM/PM (e.g., 03:05:09 PM).

Updates every second automatically.

Uses the intl package for easy formatting (no need for manual lists of months or weekdays).
